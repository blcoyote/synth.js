# Synth V2 - PC Keyboard Layout (Danish)

Complete mapping for 37 piano keys (C2 to C5) using Danish keyboard layout.
**Matches the V1 synth exactly.**

## Layout Overview

The keyboard uses 2 main rows of your Danish keyboard:

- **Bottom Row** (<AZSX...): C2 to F3 (18 keys, missing F#3)
- **Top Row** (QWERTY...): G3 to C5 (19 keys)

## Complete Key Mapping

### C2 Octave (Bottom Row Start)
```
White Keys:  <    Z    X    C    V    B    N    M
Notes:      C2   D2   E2   F2   G2   A2   B2   C3

Black Keys:  A    S    F    G    H
Notes:      C#2  D#2  F#2  G#2  A#2
```

### C3 Octave (Bottom Row Continues)
```
White Keys:  M    ,    .    -
Notes:      C3   D3   E3   F3

Black Keys:  K    L
Notes:      C#3  D#3

Note: F#3 has no key assignment (matches V1)
```

### G3 to C5 (Top Row - QWERTY + Numbers)
```
White Keys:  Q    W    E    R    T    Y    U    I    O    P    Å
Notes:      G3   A3   B3   C4   D4   E4   F4   G4   A4   B4   C5

Black Keys:  2    3    5    6    8    9    0
Notes:      G#3  A#3  C#4  D#4  F#4  G#4  A#4
```

## Features

✅ **All 37 keys have PC keyboard assignments** - No gaps!
✅ **Visual feedback** - Keys light up when pressed (both mouse and keyboard)
✅ **Logical layout** - Follows piano keyboard layout (white keys = letters, black keys = numbers/symbols)
✅ **No key repeat** - Holding a key down doesn't retrigger the note
✅ **Window blur protection** - All notes released when window loses focus

## Tips

- The layout roughly follows a piano's layout with white keys on letters and black keys on numbers/symbols
- Press multiple keys simultaneously for chords
- The visual keyboard shows labels for each key
- Keys visually press down when activated from either mouse or keyboard
